# Projekt Vertex

Tohle je můj projekt vlastního programovacího jazyka. Jmenuje se Vertex. 
Chtěl jsem udělat něco, co vypadá jako JavaScript, ale je to jednodušší.

## Co to umí
- Můžete si vytvořit proměnnou přes `var`.
- Můžete měnit hodnotu proměnné.
- Program umí vypsat výsledek přes `print`.

## Jak to spustit
1. Musíte mít nainstalovaný Python.
2. Spusťte soubor `main.py` ve složce `src`.
3. Otevře se okno, tam napíšete kód.
4. Kliknete na tlačítko RUN a uvidíte výsledek dole v černém poli.

## Příklad kódu
